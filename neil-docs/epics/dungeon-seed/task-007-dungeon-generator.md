# Task 007 — Implement procedural dungeon generator and room templates

Complexity: 13
Dependencies: Tasks 2,3

Description:
Build the deterministic dungeon generation engine driven by seed metadata and mutation state. Implement room templates (combat, treasure, trap, puzzle, rest, boss), graph connectivity, and validation tooling to check navigability and difficulty scaling.

Acceptance Criteria:
- Deterministic generator producing reproducible layouts
- Room template library and selection logic
- Validation tests for connectivity and difficulty scaling
